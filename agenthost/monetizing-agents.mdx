## Setting Up Monetization

### Basic Configuration

Configure your agent's monetization settings through the dashboard:

```json Monetization Configuration
{
  "monetization": {
    "enabled": true,
    "model": "subscription",
    "pricing": {
      "free_tier": {
        "messages_per_month": 100,
        "features": ["basic_chat", "text_only"]
      },
      "pro_tier": {
        "price": 29.99,
        "currency": "USD",
        "billing_cycle": "monthly",
        "messages_per_month": 1000,
        "features": ["advanced_chat", "file_uploads", "api_access"]

      "enterprise_tier": {
        "price": 299.99,
        "currency": "USD",
        "billing_cycle": "monthly",
        "messages_per_month": "unlimited",
        "features": ["all_features", "priority_support", "custom_branding"]
      }
    }
  }
}
```

## Pricing Strategies

### Tiered Pricing

Design pricing tiers that grow with your customers:

<AccordionGroup>
  <Accordion icon="star" title="Free Tier">
    **Purpose**: User acquisition and product validation
    
    **Typical limits**:
    - 50-100 messages per month
    - Basic features only
    - Community support
    - AgentHost branding
    
    **Strategy**: Provide enough value to demonstrate potential while encouraging upgrades
  </Accordion>
  
  <Accordion icon="rocket" title="Pro Tier">
    **Purpose**: Individual users and small teams
    
    **Typical features**:
    - 1,000-10,000 messages per month

    - Custom branding options
    
    **Price range**: $9.99 - $49.99/month
  </Accordion>
  
  <Accordion icon="building" title="Enterprise Tier">

    **Price range**: $99.99 - $999.99/month

</AccordionGroup>
